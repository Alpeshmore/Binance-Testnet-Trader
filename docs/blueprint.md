# **App Name**: Binance Testnet Trader

## Core Features:

- API Credential Validation: Validate API keys to ensure they are correct.
- Market Order Placement: Place market orders on the Binance Futures Testnet for USDT-M contracts.
- Limit Order Placement: Place limit orders on the Binance Futures Testnet for USDT-M contracts.
- Order Input Validation: Validate order parameters to prevent errors during submission. 
- Execution status and logging: Log details for debugging.
- Buy and sell execution: Add execution details and options for Buy and sell side.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to convey a sense of digital technology.
- Background color: Dark charcoal (#222E3A) for a modern, serious feel.
- Accent color: Lime Green (#32CD32) to highlight key actions or information.
- Font: 'Inter' sans-serif font for clear readability, for headlines and body.
